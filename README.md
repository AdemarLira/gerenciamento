# Gerenciamento

## Descrição
- Sistema de gerenciamento de uma clínica

## Utiliza:

- javascript
- html e  estilização em css
- php
- banco de dados (MYSQL)

### Interface em HTML:
Um formulário de cadastro com os seguintes campos:
- **Nome** (texto)
- **Email** (email)
- **Senha** (númerico)
- **Botão para enviar**
